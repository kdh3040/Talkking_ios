톡킹 이용안내
톡킹 메인화면에는 인기순(선물 받은 하트 순), 팬보유 순, 가까운 순, NEW(신규 가입순) 으로 사용자가 표시됩니다
이름, 위치, 나이를 확인하고 상대에게 쪽지나 하트를 보낼 수 있습니다.
메뉴 별 이용안내
1. 톡킹 메인
[홈]
인기순, 팬보유순, 가까운순, New를 선택하여 사용자를 볼 수 있습니다.

        (1) 인기순
        선물 받은 하트 순이 많은 인원이 표시됩니다

        (2) 팬보유순
        가입된 사용자들 중에 보유하고 있는 팬의 수가 많은 순으로 표시됩니다

        (3) 가까운순
        가입된 사용자들 중에 본인 위치와 가까운 상대가 표시됩니다

        (4) New
        가입된 사용자들 중에 신규 가입자가 표시됩니다


[즐겨찾기]
즐겨찾기로 등록한 상대나 하트를 선물 한 상대가 표시됩니다
        즐겨찾기 목록을 클릭 할 경우 해당 페이지로 이동합니다
        특정 목록을 길게 누를 경우 삭제 팝업이 표시됩니다

[채팅]
현재 생성되어 있는 채팅 목록이 표시됩니다
        채팅 목록을 클릭 할 경우 해당 채팅방으로 이동합니다

       1.채팅방
        좌측 상단에 채팅하고 있는 상대의 닉네임이 표시됩니다

        (1) 채팅 작성
        하단의 텍스트창에서 100자 이내로 메세지를 작성 할 수 있습니다
        (2) 전송
        우측 하단의 전송 버튼으로 작성된 메세지가 전송됩니다
        (3) 첨부
        좌측 하단의 + 모양 버튼으로 사진 또는 하트를 전송 할 수 있습니다
        (4) 신고
        우측 상단의 사이렌 모양 버튼으로 상대방을 신고 할 수 있습니다
        (5) 차단
        우측 상단의 손바닥 모양 버튼으로 상대방을 차단 할 수 있습니다

[내 팬]
본인에게 하트를 선물한 상대가 표시됩니다
팬 목록을 클릭 할 경우 해당 페이지로 이동합니다

[게시판]
        사용자 사진, 이름, 계급, 보석, 본문, 작성 날짜가 표시됩니다
우측 상단의 연필 모양 아이콘을 클릭하여 게시글을 등록 할 수 있습니다
        도배 방지를 위해 10분에 1개의 게시글을 등록 할 수 있습니다
        게시글을 클릭 할 경우 해당 페이지로 이동합니다
        우측 상단의 내글 아이콘을 클릭하여 작성한 글을 관리 할 수 있습니다


        [검색 필터]
상단 돋보기 아이콘을 클릭하여 검색 필터를 설정 할 수 있습니다.
        표시 개수는 메인 화면의 가로 1줄에 표시되는 상대방의 최대 사진 개수를 의미합니다
        표시 성별은 메인 화면에 표시되는 상대방의 성별을 의미합니다
        표시 나이는 메인 화면에 표시되는 상대방의 나이를 의미합니다

        [내 보석]
현재 가지고 있는 보석이 표시됩니다
        우측 상단에 보유 코인이 표시됩니다
        하단 버튼을 클릭하여 보석을 구매할 수 있습니다

        [하트아이콘]
        상단 하트 아이콘을 클릭하면 최근에 받은 하트를 확인 할 수 있습니다.
        특정 항목을 클릭 하여 내용을 확인하거나 삭제 할 수 있습니다
        우측 상단 메뉴를 사용하여 전체 리스트를 삭제 할 수 있습니다
        [내 정보]
        상단 최우측 설정 아이콘을 클릭하면 내 정보를 확인 할 수 있습니다.
        본인의 사진, 닉네임, 계급, 보석, 코인이 표시됩니다
        프로필 수정 버튼을 클릭하여 정보를 수정 할 수 있습니다
        설정 버튼을 클릭하여 톡킹 설정을 할 수 있습니다


        (1) 프로필 수정
        본인 사진을 최대 4장까지 등록 할 수 있습니다
        닉네임을 변경 할 수 있습니다
        닉네임 변경은 최초 1회는 무료이지만 그 이후는 50코인이 필요합니다
        나이를 변경 할 수 있습니다
        성별을 변경 할 수 있습니다
        하고싶은 말을 100자 이내로 변경 할 수 있습니다
        저장 버튼을 통해 변경된 프로필을 저장 할 수 있습니다

        (2) 설정
        알림 설정 기능은 쪽지, 선물, 채팅이 수신 되었을 때 소리 혹은 진동의 발생 유무를 의미합니다
        수신 거부 기능은 쪽지 수신을 거부하는 것을 의미합니다. 선물은 거부 할 수 없습니다
        차단 목록 관리 기능은 본인이 차단한 상대방의 목록을 확인하고 관리 할 수 있습니다
        톡킹 공유 기능은 톡킹을 SMS 혹은 메신저를 통해 공유 할 수 있습니다
        코인 구매 기능은 톡킹에서 사용되는 코인을 구매 할 수 있는 화면으로 이동합니다
        이용 약관 기능은 톡킹의 이용 약관을 확인 할 수 있는 화면으로 이동합니다
        도움말 및 고객지원 기능은 톡킹의 도움말 및 고객지원을 확인 할 수 있는 화면으로 이동합니다
        계정 삭제 기능은 본인의 정보가 삭제되며 재 가입화면으로 이동합니다
        복구가 불가능하오니 계정 삭제는 유의하시기 바랍니다


        [상대방 페이지]
        상대방 페이지에서는 사진, 닉네임, 나이, 계급, 보석, 보유 팬이 표시됩니다
        사진을 클릭하여 상대방이 등록한 사진을 확인 할 수 있습니다

        상대방이 보유하고 있는 팬이 있을 경우에 팬이 표시됩니다

        하단의 별 아이콘을 클릭하여 본인의 즐겨찾기 페이지로 등록 할 수 있습니다
        하단의 쪽지 아이콘을 클릭하여 상대방에게 쪽지를 보낼 수 있습니다
        (채팅방이 생성된 상대일 경우 채팅방으로 이동합니다)
        하단의 하트 아이콘을 클릭하여 상대방에게 하트를 선물하고 팬으로 등록 될 수 있습니다
        하단의 공유 아이콘을 클릭하여 메신저 혹은 SMS로 상대방을 공유 할 수 있습니다



이용제제 및 결제관련 상담은 dohosoft@gmail.com 으로 문의가 가능합니다.
